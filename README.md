# Assembly MIPS - Sequência de Fibonacci

## 📖 Objetivo

Implementar a sequência de Fibonacci utilizando Assembly MIPS, aplicando conceitos fundamentais da programação em baixo nível.

## 📂 Conteúdo

O projeto aborda:

- Sequência de Fibonacci;
- Estruturas de repetição;
- Operações aritméticas;
- Manipulação de registradores.

## 💻 Tecnologias

- Assembly MIPS
- MARS Simulator

## ▶️ Como executar

1. Abra o arquivo no MARS Simulator.
2. Monte o programa (Assemble).
3. Execute o código (Run).
